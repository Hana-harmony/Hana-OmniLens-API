# ADR-001 서비스 경계

## 결정
Hana-OmniLens-API는 한국 주식 시장 데이터, 매매제한 판단 데이터, 뉴스·공시 인텔리전스, 세무 환급 상태를 협력사에 제공하는 B2B API를 담당한다.

## 이유
- 실제 주문, 체결, 정산, 환전은 국내 증권사와 현지 브로커 인프라 영역이다.
- 협력사 거래소는 이 API를 사용해 모의 주문, 사용자 보유종목, watchlist, 알림함, 푸시 발송, 세무 서류 업로드 UI를 자체 관리한다.
- 최신 기능정의는 주문 화면 지원, 뉴스·공시 인텔리전스, 세무 환급 상태를 하나의 서비스 경험으로 묶지만, 사용자별 실행과 계정 관리는 협력사 시스템에 남긴다.

## 영향
- 이 레포는 실제 거래 명령 API를 제공하지 않는다.
- 주문 관련 API는 외국인 투자 한도, 당일 예측 지분율 boundary, VI 발동, 상·하한가 상태, 모의 주문 가능성 판단을 제공한다.
- 알림 수신자 매칭, 푸시 발송, 사용자 알림함 저장은 협력사 백엔드 책임이다.
- 세무 서류 파일 업로드 UI와 최종투자자 계정별 표시 상태는 협력사 FE/BE 책임이고, Hana-OmniLens-API는 검증/판정/환급 상태 계약을 제공한다.
