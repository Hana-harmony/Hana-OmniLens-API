# ADR-001 서비스 경계

## 결정

Hana-OmniLens-API는 한국 주식 시장 데이터, 매매제한 판단 데이터, 뉴스·공시 인텔리전스를 협력사에 제공하는 B2B API를 담당한다.

## 이유

- 실제 주문, 체결, 정산, 환전은 국내 증권사와 현지 브로커 인프라 영역이다.
- 협력사 거래소는 이 API를 사용해 모의 주문, 사용자 보유종목, watchlist, 알림함, 푸시 발송을 자체 관리한다.
- AI 모델 학습과 추론은 Hannah-Montana-AI에 두고, 이 레포는 provider 수집·정규화·전송 계약에 집중한다.

## 영향

- 이 레포는 실제 거래 명령 API를 제공하지 않는다.
- 주문 관련 API는 외국인 투자 한도, 당일 예측 boundary, VI, 단일가, 상·하한가, 거래정지 신호를 제공한다.
- 알림 수신자 매칭, 푸시 발송, 사용자 알림함 저장은 협력사 백엔드 책임이다.
